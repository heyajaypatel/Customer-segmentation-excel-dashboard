# 📊 Market Research & Customer Segmentation Tool (Excel)

## 🎯 Objective
To analyze customer data and segment customers based on behavior, spending, and engagement to support data-driven marketing decisions.

---

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts & Dashboards  
- Advanced Formulas  
- Conditional Formatting  

---

## 📂 Dataset Overview
The dataset includes 100+ customer records with attributes such as:
- Demographics (Age, Gender, City, Income)  
- Purchase behavior (Frequency, AOV, Total Spending)  
- Engagement metrics (Customer type, Discount sensitivity)  
- Product & channel preferences  

---

## 📊 Key Features
- Customer segmentation (VIP, Premium, Budget)  
- Revenue and behavior analysis  
- Interactive dashboard with slicers  
- Region, channel, and category insights  
- KPI-based performance tracking  

---

## 📈 Key KPIs
- Total Customers  
- Total Revenue  
- Average Order Value (AOV)  
- High-Value Customers  
- Repeat Customer %  
- Segment-wise Revenue
